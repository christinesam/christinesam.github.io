MY PROJECTS

Starry Starry Night

Starry Starry Night is a simple 2D Android game built using Unity's game development platform. Using the phones built in accelerometer, the player must move their boat to catch all the falling stars.

Mind the Gap

Mind the gap is an Android Application that maps the locations of stop and tube lines on the London Underground transit system. It also parses JSON files to retrieve real time arrival information of stops.
